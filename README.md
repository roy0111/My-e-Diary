# My-e-Diary



#### What the project should do
* User can register
* User can save his/her important information
* User can add his friend in contact list
* User can set event which Notify on time 
* Record your daily experience

#### Tools Used:

* PHP 5.6
* MySQL
* jQuery
* html
* bootstarp
* Ampps
